## Processing Guide

When processing inbox items, ask:
1. **Is it actionable?**
   - Yes → Move to Open Tasks or appropriate location (goals, initiatives, person files)
   - No → Reference material or delete

2. **Is it urgent?**
   - Yes → Add to today's daily note
   - No → Keep in Open Tasks

3. **Who owns it?**
   - Me → Open Tasks
   - Team → Delegate and track
   - Not me → Forward or decline

4. **What type is it?**
   - Task → Open Tasks or initiative tracker
   - Idea → `frameworks/` or `initiatives/`
   - Decision → `decisions/`
   - Person context → `people/@Name.md`
